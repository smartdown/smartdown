
##### Simple Examples

Here is a P5JS circle:


```p5js/playable/autoplay/inline
p5.setup = function() {
};

p5.draw = function() {
  p5.ellipse(50, 50, 80, 80);
};
```

and here is a rectangle abutted with math $E=mc^2$


```p5js/playable/autoplay/inline
p5.setup = function() {
};

p5.draw = function() {
  p5.rect(5, 5, 80, 80);
};
```


Pretty cool, huh?

So you can see how the circle and rectangle above were rendered inline with the text.
